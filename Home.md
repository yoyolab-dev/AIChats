## 项目简介
AIChats 是一个基于 AI 的聊天平台，使用以下技术栈：
- 前端: Vue 3 + Naive UI + Vite
- 后端: Fastify + PostgreSQL
- 部署: Docker + Kubernetes + Traefik
- 域名: pm.oujun.work (前端), api.oujun.work (后端)
